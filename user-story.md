# User Story Template

**Role:** As a User  
**Action:** I want to manage my account details  
**Benefit:** So that I can keep my information up to date.

---

## User Stories
1. **Create Account:** As a user, I want to create an account so that I can access the service.
2. **Read Account:** As a user, I want to see my account details so that I can verify my info.
3. **Update Account:** As a user, I want to update my email so that I receive notifications.
4. **Delete Account:** As a user, I want to delete my account so that my data is removed.
5. **List Accounts:** As an admin, I want to list all accounts to monitor usage.
